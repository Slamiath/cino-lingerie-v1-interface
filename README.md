# cino-lingerie-v1-interface
Ce projet est un exercice pratique dédié à la maîtrise du positionnement moderne avec Flexbox CSS. L'objectif était de reproduire une interface de catalogue élégante pour une marque de lingerie.

# Objectifs
Maîtriser l’alignement horizontal et vertical des éléments avec justify-content et align-items.
Comprendre la distribution de l'espace avec la propriété gap.
Manipuler les conteneurs flexibles pour créer un header, un catalogue et un footer structurés.

# Technologies utilisées
HTML5 : Structuration sémantique de la page.
CSS3 : Utilisation exclusive de Flexbox (sans frameworks).
